**Exercício 01: SINTAXE JAVA / ESTRUTURA DE CONDIÇÃO**



Faça um programa que mostre o menu de opções a seguir, receba a opção do usuário e os dados necessários para executar cada operação. Obrigatório o uso do switch/case.

Menu de opções:

&#x09;1. Somar dois números.

&#x09;2. Subtrair dois números

&#x09;3. Multiplicar dois números

&#x09;4. Dividir dois números

&#x09;5. Raiz quadrada de um número

&#x09;Digite a opção desejada.



**Exercício 02: ESTRUTURA DE REPETIÇÃO**



&#x09;1. Implemente um programa que recebe repetidamente um número de 1 a 12, enquanto esse número for diferente de 0 (zero), e imprime o mês correspondente. Quando o número estiver fora do intervalo permitido, a mensagem “mês inválido” deverá ser exibida.



&#x09;2. Faça um programa que receba a idade de 10 pessoas e que calcule e mostre a quantidade de pessoas com idade maior ou igual a 18 anos. Obrigatório o uso da estrutura for.



**Exercício 03: ARRAYS JAVA**



&#x09;1. Elabore um programa que preencha um array de 10 posições digitados pelo usuário(valores inteiros), ao final o programa deve subtrair 1 dos números impares e adicionar 1 aos números pares. Exibir em tela os novos valores do array.



&#x09;2. Elabore um programa que preencha um array de 10 posições digitado pelo usuário(valores inteiros), calcule e mostre:

&#x09;	a) A quantidade de números pares.

&#x09;	b) Quais os números pares.

&#x09;	c) A quantidade de números ímpares.

&#x09;	d) Quais os números ímpares
