# Técnicas de Programação I – Atividades em Java

Repositório com exercícios desenvolvidos na disciplina **Técnicas de Programação I – Ciclo 02**.

O objetivo deste repositório é armazenar e organizar as atividades práticas realizadas durante as aulas utilizando a linguagem **Java**, com foco no desenvolvimento da lógica de programação e no aprendizado dos conceitos fundamentais da linguagem.

## Estrutura do repositório

Cada exercício está organizado em **uma pasta própria**.
Dentro de cada pasta existe um **package Java** contendo as classes e o código da atividade.

Exemplo de organização:

```
exercicio_01/
 └── src/
      └── edu.fatec.tecprog.exercicio_01
           └── Main.java

exercicio_02/
 └── src/
      └── edu.fatec.tecprog.exercicio_02
           └── Main.java
```

Cada exercício possui uma classe `Main` responsável pela execução do programa.

## Conteúdos trabalhados

* Estrutura básica de programas em Java
* Variáveis e tipos de dados
* Estruturas condicionais
* Estruturas de repetição
* Vetores
* Entrada e saída de dados
* Organização de código em classes e pacotes
